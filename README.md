
# Universidade Weitinho
Uma aplicação para gerenciar o banco de dados de uma universidade, que armazena alunos, professores, disciplinas, turmas e gerentes.

Feito para a tarefa final da disciplina de Programação Orientada à Objetos, no curso de Ciência da Computação da Universidade Federal Rural do Semi-Árido.

## Deployment
O programa necessita do banco de dados aberto para funcionar.

Para iniciar o banco de dados, tenha o Docker (ou uma alternative como o Rancher) aberto, e utilize o comando abaixo no diretório raiz.
```bash
  docker compose up
```

Abra o projeto no IntelliJ IDEA, selecione e execute o arquivo java no diretório abaixo.
```
src/main/java/br/edu/ufersa/universidade/Main.java
```
